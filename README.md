# Netball Super League Malaysia Analytics Dashboard

## Project Description
A responsive web application that displays NSL Malaysia 2026 season statistics and performance data.

## Features Included
- Login and registration interface
- Dashboard with season statistics
- Interactive charts
- Top players and team lineups
- Match results
- League standings
