# Netball Super League Malaysia Analytics Dashboard

## Project Description
The Netball Super League Malaysia Analytics Dashboard is a responsive web application that presents information about the NSL Malaysia 2026 season. It allows users to explore team information, player performance, match results, league standings, and season statistics through interactive data visualisations.

## Features Included
- User login and registration interface
- Dashboard with season summaries and statistics
- Interactive bar chart and pie chart
- NSL team information
- Top player performance and team lineups
- Complete match results
- League standings and win percentages
- Responsive navigation and mobile-friendly design
